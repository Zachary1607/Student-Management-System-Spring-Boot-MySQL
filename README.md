# Student Management System

A web-based Student Management System built using Spring Boot and MySQL. The application provides functionality to create, view, update, search, and delete student records through a clean web interface.

## Features

- Add Student Records
- Update Student Information
- Delete Student Records
- Search Students
- Form Validation
- MySQL Database Integration
- Layered Spring Boot Architecture

## Tech Stack

- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- MySQL
- Thymeleaf
- Maven

## Project Structure

src/main/java
├── controller
├── service

└── StudentManagementApplication

src/main/resources
├── templates
├── static
└── application.properties
